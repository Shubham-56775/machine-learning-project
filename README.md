# machine-learning-project# 
🩺Diabetes Prediction using Machine Learning

## 📌 Project Description

This project uses Machine Learning to predict whether a person has diabetes or not based on medical features.
The model is built using Logistic Regression.

---

## 🎯 Problem Statement

The goal of this project is to develop a predictive model that can classify patients as diabetic or non-diabetic using given health-related data.

---

## 📊 Dataset Information

* Dataset: Diabetes Dataset (CSV file)
* Features include:

  * Pregnancies
  * Glucose
  * Blood Pressure
  * Skin Thickness
  * Insulin
  * BMI
  * Diabetes Pedigree Function
  * Age
* Target:

  * Outcome (0 = Non-diabetic, 1 = Diabetic)

---

## 🧠 Model Used

* Logistic Regression

---

## ⚙️ Technologies Used

* Python
* Pandas
* Scikit-learn

---

## 🔍 Steps Performed

1. Loaded dataset using Pandas
2. Explored dataset (shape, description, value counts)
3. Separated features and target variable
4. Split data into training and testing sets
5. Trained Logistic Regression model
6. Evaluated model using accuracy score
